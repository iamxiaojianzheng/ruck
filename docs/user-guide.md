# Ruck 用户指南

欢迎使用 Ruck！本文档将帮助您快速上手 Ruck。

##  快速开始

### 安装

从 [GitHub Releases](https://github.com/iamxiaojianzheng/ruck/releases) 下载适合您系统的安装包。

- **macOS**: `.dmg` 文件
- **Windows**: `.exe` 安装程序
- **Linux**: `.deb` 包

### 首次启动

安装完成后，首次启动会显示引导页面。

**默认快捷键**：`Alt + R`（Windows/Linux）或 `Option + R`（macOS）

---

## 🔍 基本功能

### 搜索应用

按下快捷键呼起主窗口，直接输入应用名称即可搜索。

**支持**：
- 完整名称搜索
- 拼音搜索
- 拼音首字母搜索

**示例**：
- 输入 `vscode` 或 `vs` 搜索 Visual Studio Code
- 输入 `weixin` 或 `wx` 搜索微信

### 搜索插件

输入插件的触发命令搜索插件。

**示例**：
- 输入 `calc` 打开计算器插件
- 输入 `trans` 打开翻译插件

---

## 🔌 插件管理

### 安装插件

1. 点击搜索框右侧的 Ruck 图标
2. 进入插件市场
3. 浏览或搜索插件
4. 点击"下载"按钮安装

### 卸载插件

1. 进入插件市场
2. 切换到"已安装"标签
3. 找到要卸载的插件
4. 点击"卸载"按钮

### 使用插件

在主窗口搜索框中输入插件的命令关键词，选择插件即可打开使用。

---

## ⚙️ 设置

在主窗口搜索"偏好设置"进入设置页面。

### 快捷键设置

可以自定义以下快捷键：
- 主窗口显示/隐藏
- 截图
- 分离窗口

### 主题设置

支持明亮和暗黑两种主题。

### 开机启动

可以设置 Ruck 开机自动启动。

---

## 💾 数据同步

### 配置 WebDAV

1. 进入"偏好设置"
2. 选择"账户和设置"
3. 配置 WebDAV 服务器地址、用户名和密码
4. 点击"测试连接"

### 导出数据

点击"导出数据"将插件配置和历史记录上传到 WebDAV 服务器。

### 导入数据

在新设备上配置相同的 WebDAV 设置，点击"导入数据"同步数据。

##❓ 常见问题

### 快捷键不生效？

请检查：
1. 快捷键是否与其他应用冲突
2. Ruck 是否在后台运行
3. 尝试重启 Ruck

### 插件无法安装？

请检查：
1. 网络连接是否正常
2. npm registry 是否可访问
3. 查看错误日志

### 数据同步失败？

请检查：
1. WebDAV 配置是否正确
2. 网络连接是否正常
3. 服务器是否可访问
